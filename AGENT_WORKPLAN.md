# TruckFixr Agent Workplan

This workplan turns the current module goals into concrete work the different agents can do together.

## Marketing Agent

Objective: Shape positioning and demand experiments around the jobs-to-be-done that move real buyers.

Work for the next sprint:

- Publish the 5 seeded content ideas across LinkedIn, email, and proof-led nurture assets.
- Turn one pilot conversation into a founder post, one funding-credibility post, and one partner enablement email.
- Build a simple weekly content loop: topic, audience, proof point, CTA, owner, publish date.
- Report which messages create the most replies, demo requests, and support-letter conversations.

Outputs:

- 5 content assets
- 1 weekly content calendar
- 1 message-performance summary

## Sales Agent

Objective: Feed qualified fleet opportunities into pilots, proof, and support-letter conversations.

Work for the next sprint:

- Prioritize 15 Ontario fleets most likely to have maintenance coordination pain.
- Tag prospects by fleet size, downtime pain, pilot fit, and support-letter likelihood.
- Run outreach that asks for a short discovery call or a structured feedback session.
- Flag every prospect that could strengthen OVIN, Mitacs, or future grant applications.

Outputs:

- Top-15 target list
- 5 discovery conversations booked
- 3 support-letter candidates

## Funding/R&D Agent

Objective: Keep funding research disciplined and convert it into application-ready work.

Work for the next sprint:

- Triage the 5 seeded opportunities into `apply now`, `partner first`, and `watch`.
- Complete grant-readiness checklists for OVIN, Mitacs, SR&ED, C2C, and Ready 4 Market.
- Draft one project concept note for OVIN or Mitacs and one documentation checklist for SR&ED.
- Coordinate missing dependencies with sales, partnerships, evidence, and engineering.

Outputs:

- Ranked funding pipeline
- 2 concept notes
- 1 SR&ED documentation checklist

## Pilot Evidence Agent

Objective: Gather proof that makes the product easier to buy, fund, and expand.

Work for the next sprint:

- Capture 3 high-quality evidence entries from pilot or discovery conversations.
- Standardize each entry into problem observed, technical uncertainty, experiment, result, and commercialization relevance.
- Build a quote bank with approved anonymized customer language.
- Mark which evidence can support content, support letters, and funding claims.

Outputs:

- 3 evidence records
- 1 reusable quote bank
- 1 list of proof points by audience

## Partnerships Agent

Objective: Open channels that increase reach and unlock co-development funding.

Work for the next sprint:

- Identify one Ontario academic lab and one ecosystem partner aligned to fleet AI or mobility workflows.
- Prepare a one-page partner brief explaining the problem, project idea, and expected commercial upside.
- Start conversations that can support Mitacs, C2C, or OVIN partnership requirements.
- Track integration or data-sharing dependencies that affect future pilots.

Outputs:

- 2 partner briefs
- 2 active partner conversations
- 1 dependency list for co-development projects

## Engineering Agent

Objective: Support product delivery and evidence capture without losing focus on customer value.

Work for the next sprint:

- Make it easier to export funding opportunities, R&D evidence, and proof assets into one packet.
- Add lightweight tagging for proof points that are safe for marketing, funding, or investor use.
- Improve the reliability of the funding and evidence workflows so the operating system can be used in live founder rhythm.
- Support instrumentation that captures technical uncertainty and experiment history for SR&ED.

Outputs:

- Exportable grant packet workflow
- Proof-tagging support
- SR&ED-friendly evidence capture improvements

## Shared Weekly Rhythm

- Monday: Funding Agent reviews deadlines and missing dependencies.
- Tuesday: Sales and Partnerships Agents update support-letter and partner pipeline.
- Wednesday: Evidence Agent publishes new proof points for reuse.
- Thursday: Marketing Agent ships one asset using fresh evidence.
- Friday: Engineering Agent exports the latest funding-evidence packet and logs gaps.
